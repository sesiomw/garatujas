# garatujas do Moisés IA25

Garatujas são os primeiros rabiscos e expressões gráficas espontâneas de crianças, geralmente entre 0 e 6 anos, representando uma etapa fundamental para o desenvolvimento da coordenação motora fina, criatividade e comunicação pré-verbal. Esses traços, que evoluem com o tempo, permitem que a criança explore o papel, registre emoções e inicie sua representação do mundo.

# TypeScript

## POO:

é a abstração do mundo real.
ajuda a organizar o código por partes e de maneira reutilizável,
facilita o desenvolvimento e a manutenção do sistema.

## PE vs POO:

a programação estruturada consiste em criar funções e procedimentos para
manipulação de dados. Já a programação orientada a objetos consiste em criar
objetos que combinem atributos e métodos.

## Classe e objeto:

classe é o molde que define atributos e métodos específicos de um objeto.
serve como um modelo para criação de objetos.

objeto é a instância da classe.
possui atributos e métodos definidos pela classe.
cada objeto pode ter valores diferentes para os atributos.

## .this:

o this é usado dentro de classes para referenciar o objeto atual.
a instância que está sendo manipulada no momento.
permite acessar atributos e métodos do objeto dentro da definição da classe.
 
## Construtor:

é o método especial de uma classe chamado quando o objeto é criado.
usado para inicializar atributos do objeto podendo receber parâmetros para
configurar o estado inicial do objeto.
em TypeScript o consturor é definido usando a palavra constructor.
pode ser sobrecarregado para permitir diferentes formas de criar objetos.

## Encapsulamento:

garante a abstratividade e segurança dos dados.
consulta em ocultar detalhes internos de uma classe e expor apenas o
necessário para o uso externo.
é feito utilizando modificadores de acesso como private, protected e public
